```
                           |
          Osfin     Zeni     ChatFin
          (recon)   (AI agents) (AI agents)
                           |
                    EXECUTION / OPS
```

### The Massive Gap OnEasy Fills

| Capability | Existing Players | OnEasy AI |
|---|---|---|
| AI Agents for Finance | ChatFin, Zeni, Datarails, Cube, Vena | YES |
| Execution (bookkeeping, ops) | Pilot, Zeni, BlackLine | YES |
| Advisory (CFO-level strategy) | Kruze, Graphite, Pilot (all human) | YES (AI-powered) |
| Compliance & Regulatory | MindBridge (audit only) | YES (full compliance) |
| Capital Structuring | NOBODY | YES |
| India-First / Emerging Markets | Osfin (reconciliation only) | YES |
| All-in-One Platform | NOBODY combines all above | YES |

### Key Insight
**No single competitor combines AI agent automation + execution + advisory + compliance + capital structuring.** They are all siloed:
- Category 1 (ChatFin/Datarails/Cube/Vena) = FP&A automation only
- Category 2 (BlackLine/MindBridge/Osfin) = Back-office ops only  
- Category 3 (Kruze/Pilot/Graphite) = Human services only (expensive, doesn't scale)
- Category 4 (AlphaSense/Hebbia) = Intelligence/research only

**OnEasy AI is the only platform that can sit at the intersection of ALL FOUR categories** — with an India-first go-to-market that none of these competitors address.
